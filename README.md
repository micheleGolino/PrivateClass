# Repository PrivateClass
<h1><b>Classi private</b></h1>

<h2>In questo progetto si utilizza la JDK 16 di Eclipse standard:</h2>

<h3>
	<ul>
		<li>
			<b>CHttpClient.java</b> - Classe Wrapper che sostituisce la classica <a href="https://docs.oracle.com/en/java/javase/11/docs/api/java.net.http/java/net/http/HttpClient.html">Class HttpClient</a>;
		</li>
		<li>
			<b>PostCall.java - Esegue una chiamata POST, in cui è possibile passargli diversi parametri;</b>
		</li>
		<li>
		<b>GetCall.java - Esegue una chiamata GET, in cui è possibile passargli diversi parametri;</b>
		</li>
		<li>
		<b>Utility.java - Contiene dei metodi sulle mappe e sulle conversioni in JSON.</b>
		</li>
	</ul>
</h3>
